# Galib Font

A handcrafted pixel art font designed for retro-style games, UI systems, and creative coding projects.

---

## ✨ Features

- 🟩 Pixel-perfect  
- 🎮 Ideal for retro & indie games  
- 🖥️ Great for terminal-style UI  
- 📦 Lightweight and clean design    

---

## 🚀 Usage

### Install (Desktop)

1. Download the font file from the repository.
2. Install it on your system.
3. Use it in your design or development tools.


---

## 🌐 Web Usage

```css
@font-face {
  font-family: 'galib-font';
  src: url('./galib-font.ttf') format('truetype');
}

body {
  font-family: 'galib-font';
}
```
